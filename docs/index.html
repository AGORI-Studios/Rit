Rit.

What is Rit?

Rit is a mania-styled rhythm game created in the Love2D game framework. It is currently in development.

How do I play?

Download the latest version of Rit from the releases page. Unzip the file and run the executable.

Is my device supported?

Rit runs on majority of devices, such as, MacOS, Windows(x64/x32), Linux, and Nintendo Switch

Will this game ever have online support? such as leaderboards?

Yes, I have planned for online support in Rit, but it will not be available until the game is out of beta.

How do I make a chart?

Currently, charting is not supported in Rit, but it will be in the future. Use other games such as Osu! or Quaver to make your charts.
